# glodap3-methods

Analysis and figures accompanying the GLODAPv3 methods paper (Humphreys et al., in prep.).

## Install the Python environment

The Python environment used for this project was set up with [Pixi](https://pixi.prefix.dev/latest/installation/), and that will be the easiest way to replicate it.

Clone the repo and from the main directory run

    pixi install

## Generating the figures

The figures can be generated by running the four Python scripts starting with `fig0` in the main directory.

Created figures are saved in the `figures` directory.

The other scripts and subdirectories contain necessary supporting scripts and imported datasets.
